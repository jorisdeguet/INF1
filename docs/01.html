Nous allons expliquer ce qu'est:

  • un langage de programmation : des ordres à l'ordinateur selon une syntaxe très rigide
  • un fichier de code source (pour nous Python) : un fichier texte qui respecte cette syntaxe très rigide
  • l'interpréteur (Python pour nous) : le programme qui est capable de lire cette syntaxe et d'exécuter les ordres, nous on appelle ça des instructions
  • une expression : c'est une partie de code source qui a une valeur. Par exemple : 3 * (4+5) est une expression dont la valeur est le nombre entier 60.
  • une instruction : un ordre qu'on donne à l'ordinateur mais qui ne vaut rien. Par exemple print( 3 + 5 ) provoque l'affichage de 8 à l'écran.

On va faire une première de démo d'un script Python "Bonjour le monde".

On va montrer comment utiliser l'environnement de développement, comment démarrer un script, comment mettre un point d'arrêt.

On va discuter également des différences entre exécuter du code directement ou utiliser une fonction main.

Exercice hello-world.py

Créer un premier dossier pour vos exercices et ouvrez le dans PyCharm. Dans le projet, créez un fichier texte appelé hello-world.py.

Dans ce fichier, nous allons créer une fonction main. Cette fonction va uniquement afficher vos nom et prénom dans le console.

Exécutez cette méthode en mode normal puis en mode débogage.
