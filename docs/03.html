Exercice for-et-if.py

Faites un script Python appelé for-et-if.py.

Vous devez définir une fonction main dans votre script.

La fonction main doit parcourir les 100 000 premiers nombres entiers et afficher dans la console uniquement les nombres qui sont à la fois multiples de 3 13 et 19.
